\doxysubsection{Detailed Description}
Defines the possible types of gap penalties. This is\+: 
\begin{DoxyItemize}
\item CONSTANT, if static and the extension penalty is 0 
\item LINEAR, if static and the open penalty is 0 
\item AFFINE, if static but neither CONSTANT nor LINEAR 
\item DYNAMIC, if penalty values change during alignment 
\end{DoxyItemize}
