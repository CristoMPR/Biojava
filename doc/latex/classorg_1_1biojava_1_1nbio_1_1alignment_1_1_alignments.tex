\doxysubsection{Detailed Description}
Static utility to easily run alignment routines. To exit cleanly after running any parallel method that mentions use of the \mbox{\hyperlink{}{Concurrency\+Tools}} utility, \mbox{\hyperlink{}{Concurrency\+Tools\#shutdown()}} or \mbox{\hyperlink{}{Concurrency\+Tools\#shutdown\+And\+Await\+Termination()}} must be called.
