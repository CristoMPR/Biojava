\doxysubsection{Detailed Description}
Stores all the content of a Stockholm file. {\itshape {\bfseries{N.\+B.\+: This structure will undergo several enhancements later on. Don\textquotesingle{}t depend on it in a final code, otherwise it will be hard to maintain.}}}

In general, Stockholm File contains the alignment mark-\/up lines.~\newline
