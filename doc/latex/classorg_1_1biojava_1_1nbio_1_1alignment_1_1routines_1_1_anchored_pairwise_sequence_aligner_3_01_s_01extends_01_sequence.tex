\doxysubsection{Detailed Description}
This algorithm uses a divide-\/and-\/conquer approach to find optimal pairwise global sequence alignments (from the first until the last \mbox{\hyperlink{}{Compound}} of each \mbox{\hyperlink{}{Sequence}}) with the restriction that any alignment produced will connect the query sequence to the target sequence at the {\itshape anchors}. This class performs such global sequence comparisons efficiently by dynamic programming with a space requirement reduced from quadratic (a multiple of query sequence length times target sequence length) to only linear (a multiple of query sequence length). The counterpoint to this reduction in space complexity is a modest (a multiple \texorpdfstring{$<$}{<} 2) increase in time.
